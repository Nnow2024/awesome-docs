# 快速开始

目标：说明从拿到开发板或算力卡到跑通第一个 AI 示例的基本路径。不同芯片和板卡的固件、接口、驱动和示例包可能不同，具体操作以对应板卡资料为准。

## 5分钟点亮开发板指引

### 适用范围

| 产品 / 形态 | 当前资料状态 | 说明 |
| --- | --- | --- |
| AX8850N / AX8850N 主控开发板 | 已有示例 | 当前页面和[基础软件环境](../04_dev_environment/base_software.md)中的完整命令主要来自该类板卡 |
| AX8850 / AX8850N 算力卡 | 已有示例 | 需先安装 AXCL 驱动，再按算力卡示例运行 |
| 终端计算智能视觉芯片 | 待补充 | 需结合具体模组、开发板或整机 SDK 资料补充 |
| 其他生态板卡 | 待补充 | 由对应板卡厂家在[生态硬件](../03_hardware/index.md)章节补充 |

### 通用检查流程

1. 确认硬件型号和产品形态（主控开发板 / 算力卡 / 模组 / 整机）。
2. 查阅对应硬件页面，确认供电、串口、网络和固件资料是否齐全。
3. 查询固件版本，确认与文档中记录的验证版本一致或兼容。
4. 确认网络连通，准备模型、示例仓库和运行环境。

## 首次登录

- **主控开发板**：通常通过串口或 SSH 登录，串口参数和默认账号以板卡资料为准。
- **算力卡**：在宿主机安装 AXCL 驱动后操作，驱动安装见 [AXCL 文档](https://axcl-docs.readthedocs.io/zh-cn/latest/)。

确认网络连通：

```bash
ping www.baidu.com -c 3
```
