# NPU 示例展示（算力卡平台）

算力卡（M.2 加速卡）基于 AXCL 开发库进行 NPU 推理。完整的驱动安装、API 说明、示例代码和 Benchmark 请参阅 AXCL 专用文档：

- **中文文档**：[AXCL 使用手册（GitHub）](https://github.com/AXERA-TECH/axcl-docs)
- **英文文档**：[AXCL Documentation（ReadTheDocs）](https://axcl-docs-en.readthedocs.io/en/latest/)

文档涵盖以下内容：

- Linux 驱动安装与 AXCL-SMI 工具
- Quick Start 与 NPU Examples
- NPU Benchmark
- SDK Samples 与 API 参考
- FFmpeg 集成
- Windows 驱动安装（Beta）
- 多芯级联（TensorParallel）

```{note}
算力卡支持 x86 和 aarch64 两种 Host 平台，具体差异与编译方法见 AXCL 文档的 SDK Build 章节。
```
