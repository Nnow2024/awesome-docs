# 常见问题解答

使用过程中碰到的高频问题。欢迎通过 PR 补充。

## 环境与依赖

- **Q：PyAXEngine 从 GitHub 下载失败怎么办？**
  - A：改用 Huggingface 镜像站下载，详见 [基础软件环境](../04_dev_environment/base_software.md)。

## 硬件与固件

- **Q：如何查询开发板固件版本？**
  - A：`cat /proc/ax_proc/version`，详见 [基础软件环境](../04_dev_environment/base_software.md) 的"系统相关 / 版本查询"。
